# Telegram бот + MiniApp с автомаизированной выдачей VPN-ключей

![Unknown-4 копия](https://github.com/user-attachments/assets/018c0358-3588-46d4-b64c-bc0c7e47f904)

⚡[VPNUS](https://ishortlink.ru/r/Xe4SJw) - карманный VPN в Telegram боте и MiniApp! Управление личным кабинетом, выдача ключей и полная настройка для доступа к любимым ресурсам
Экосистема VPNUS состоит из Telegram бота, MiniApp и веб-страницы. Проект предоставляет пользователям возможность получать VPN-ключи, управлять подпиской и пополнять баланс прямо внутри Telegram любым удобным способом. Процесс от регистрации до подключение занимает не больше 1 минуты

## 🔑 Ключевые возможности:
### ⛓️‍💥 Взаимодействие пользователя с сервисом
  • Удобное отслеживание оставшегося баланса и срока окончания подписки

  • Возможность пополнения баланса на любую сумму с наглядной конвертацией в срок действия ключа с учетом выбранного     количества подключаемых устройств
  
  • Связь с технической поддержкой 24/7 для оперативного решения любого вопроса

  • Интуитивно понятное управление всеми подключенными устройствами с возможностью их принудительного отключения 
 
  • Щедрая и прозрачная реферальная программа за каждого приглашенного пользователя по ссылке

### ⛓️‍💥 Взаимодействие администратора с пользователем
  • Удобная панель, наглядно отражающая статистику и динамику всех изменений

  • Возможность блокировки пользователя по ID на любой срок, а также возможность выдачи подписки пользователю по ID также на любой срок

  • Раздел создания рассылки с поддержкой форматирования
